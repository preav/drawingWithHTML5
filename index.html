Drawing using <canvas>:

We can draw in HTML using either <canvas> or <svg> element. I'm going to use the former in this skill test project.

Drawing a Rectangle:

There are couple of methods available to draw using a canvas. But first, we need to get a reference of the <canvas> element to draw on the canvas. This is done using getContext method which accepts "2d" as parameter. Checkout default.js to see the code behind this box.

Editing the Drawing

We can also change properties of these rectangles such as fill the inside with a color other than black or change the thickness of the line when drawing a rectangle. The first rectangle is filled with a color other than black(default) using fillStyle method. For the next rectangle, we did something fancy and filled it with a "linear" gradient using createLinearGradient and addColorStop methods. The third rectangle was filled using a "radial" gradient using createRadialGradient and addColoStop methods. These CanvasGradient objects accept parameters such as x0, y0, x1, y1 and r0 and r1 for radial gradient. These parameters basically tell where the gradient should start and where it should end. You can play around the values to check how they work. Checkout default.js to see the code behind these rectangles (drawColouredRect() and drawGradientRect()).

The first rectangle experiments with lineWidth property of getContext and the second reactangle tells that we can also use gradient as borders. We've drawn the third triangle using paths. You can find the code behind these rectanges in default.js (gradientBorder() function).
